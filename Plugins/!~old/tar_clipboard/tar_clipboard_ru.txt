Название: Плагин использование буфера обмена.
Версия: 1.1
Автор: JumpingJack (jumpingjack@mail.ru)

Возможности:
Создает алиасы, копирующие строковые значение в буфер обмена.

Как настраивать:
В папке Plug-Ins должен быть файл конфига с названием tar_clipboard.ini. В него добавляются строки вида:
name|jumpingjack

где: name        - имя алиаса.
       jumpingjack - значение, которое будет помещено в буфер обмена при вызове данного алиаса.
       |                - обязательный разделитель

Требования:
1. Должен присутствовать разделитель.
2. Каждый алиас на новой строке.
3. Файл tar_clipboard.ini должне находиться  в одной папке с файлом tar_clipboard.dll

Зачем: 
Думаю плагин пригодиться для заполнения всевозможных веб-форм. Особенно удобен в связке с плагином tar_hotkeys.dll.

Примечания:
После добавления плагина необходимо перезапустить программу Type And Run.
