Название: Плагин управления winamp'ом через TypeAndRun
Версия: 1.0
Автор: Python <smisoft@rambler.ru> (c) SmiSoft (SA)

Возможности:
Позволяет примитивно управлять проигрывателем Winamp через плагинные алиасы для TypeAndRun. Разумеется, можно использовать в связке с плагином tar_hotkeys.

Алиасы:
~prev - предыдущий трек
~play - играть
~pause - пауза
~stop - стоп
~next - следующий трек
