Название: Плагин управление глобальными клавишами для TypeAndRun.
Версия: 1.0
Автор: -=GaLaN=- (Evgeniy Galantsev)

Возможности:
Позволяет создавать глобальные горячие клавиши для запуска определенных строк через TypeAndRun.

Как настраивать:

В папке с плагином должен быть файл конфига с названием tar_hotkeys.ini. В него добавляются строки вида:
mods|key|string

где:
mods - необязательные модификаторы, состоящие из четырех символов - A (Alt), C (Control), S (Shift) и W (Win) - в любом желаемом сочетании, например:
cas
w
wca

key - клавиша, используемая отдельно или вместе с модификаторами. Позволяется использовать только алфавитно-цифровые варианты плюс клавиши из следующего списка:
`
-
=
\
;
'
,
.
/
F1..F12

strings - абсолютно любая строка. Именно она по нажатию глобальной горячей клавиши будет с помощью SendMessage запущена через TypeAndRun.

Зачем это надо:
Многие просили добавить возможность в TypeAndRun запуск по горячим клавишам определенных алиасов. Однако, такая функция в корне противоречит идеологии программы TypeAndRun, призванной как для раз того, чтобы разгрузить пользователя от кучи горячих клавиш. Однако, так как число просьб сделать эту фичу все время растет, был написан плагин для достижения подобной цели.
