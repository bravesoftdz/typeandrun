Название: Различные функции для конвертации одних значений в другие.
Версия: 1.0
Автор: Юрий Чеботарев

Возможности:
1. Конвертация RGB(rrr,ggg,bbb)<->#rrggbb

Пример: введите RGB(255,204,153)
Результат будет #FFCC99

Введите #FFfa99
Результат будет RGB(255,250,153)

2. Преобразование из десятичной системы в шестнадцатиричную и обратно.

Пример преобразования из шестнадцатиричной системы в 10-ю:
Введите DEC(111)
в результате получите
273

Примечание - можно вводить и так: DEC(0x111)

Пример преобразования из шестнадцатиричной системы в 10-ю:
введите HEX(1234567689)
в результате получите 
49960209

3. Несколько вспомогательных функций.
При вводе "tm" будет выведено текущее время в формате
ЧЧ:ММ:СС

При вводе "date" будет выведена текущая дата в формате
ДД.ММ.ГГГГ

Примечание - вызовы функций конвертации регистронезависимы
Т.е. можно писать dec(... или DEC(... - рузультат один и тот же.
Аналогично регистр букв при конвертации RGB(rrr,ggg,bbb)<->#rrggbb не имеет значения.
Т.е. #FF00AA и #ff00Aa будут восприниматься одинаково.
