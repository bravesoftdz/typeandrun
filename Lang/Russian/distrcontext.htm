Содержание дистрибутива

Входящие в поставку:

TypeAndRun.exe - исполняемый файл
Lang\English.chm - английский файл помощи
Lang\Russian.chm - русский файл помощи
Lang\English.lng - модуль английского языка
Lang\Russian.lng - модуль русского языка

Появляющиеся в процессе работы:

Config.ini - конфиг для алиасов
~Config.ini - конфиг для удаленных мертвых алиасов
TypeAndRun.ini - настройки программы
History.ini - история команд
Plugins.ini - настройки плагинов

Plugins\ - рекомендуемая папка для плагинов
