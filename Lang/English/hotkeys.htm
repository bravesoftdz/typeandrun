Hot Keys

Major hot keys:

Ctrl+Z Undo function.
Ctrl+S Setup settings.
Ctrl+Q Quit the program.
Ctrl+E Config editor.
Ctrl+N Create new alias with string entered as alias action (if any entered). Otherwise creates "empty" alias for further editing.
Ctrl+A Select all text.
Ctrl+R Reread all configuration files.
Ctrl+W Clear the console window.
Ctrl+T Quick autocompletion mode switch.
Ctrl+Delete Quickly to delete the string typed in the console, from config and history.
Esc Hide the console.
F1 Open this help file.
Ctrt+Shift+Alt+LMB Move the console.
TAB Chose among the options offered by autocompletion.
Shift+TAB Chose among the options offered by autocompletion in back order.
\ Apply the specified path when using the path browser.
Enter Execute the string.
Shift+Enter (alias) Quit all processes having the file name specified in the alias settings.
Shift+Enter (URL) Start the page in the system default browser rather than one specified in the alias settings.
Shift+Space Allows you to type space in "advanced Space processing mode".
Ctrl+DownArrow Allows you to manually show the alias hint.
Alt+DownArrow Shows list of completions that will be shown while you press TAB or history.
Ctrl+D Opens entered alias/system alias target location folder in configured file manager.
Shift+Backspace Acts as regular Backspace if "advanced BackSpace processing mode" enabled.
Ctrl+Backspace Deletes all characters right to the next slash/backslash (extremely convenient if you work with paths/hyperlinks).
