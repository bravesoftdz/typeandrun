Some useful examples

1) Fast search through internet search engines

General pattern is "%alias% searched_term".
Where %alias% may be replaced with any of the following:
yandex|http://ya.ru/yandsearch?text=%s
ya|http://ya.ru/yandsearch?text=%s
google|http://www.google.com/search?q=%s&hl=en
rambler|http://search.rambler.ru/cgi-bin/rambler_search?words=%s&where=1
aport|http://sm.aport.ru/scripts/template.dll?r=%s&That=std
lycos|http://poisk.lycos.ru/cgi-bin/pursuit?query=%s&cat=loc&who=ru&x=26&y=10
Get whois information from RIPE:
RIPE|http://ripe.net/perl/whois?form_type=simple&full_query_string=&searchtext=%s
Search FTP:
FTP_search|www.filesearch.ru/cgi-bin/s?q=%s&t=f&d=&x=14&y=15
Search Moshkov library:
Moshkov|www.lib.ru/TitleSearch?Search=%s
Search iXBT hardware web portal (Russian):
ixbt_serch|http://rose.ixbt.com/cgi-bin/search.cgi?q=%s

2) Fast translation

Russian to English. General pattern is "ruen url".
ruen|http://www.translate.ru/url/tran_url.asp?lang=ru&url=%s&direction=re&template=General&cp1=NO&cp2=NO&autotranslate=on&transliterate=on&psubmit2.x=50&psubmit2.y=12
English to Russian. General pattern is "enru url".
enru|http://www.translate.ru/url/tran_url.asp?lang=ru&url=%s&direction=er&template=General&cp1=NO&cp2=NO&autotranslate=on&transliterate=on&psubmit2.x=50&psubmit2.y=12
WorldLingo Translate - Chinese Simplified to English.
General pattern is "chen url".
chen|http://www.worldlingo.com/wl/mstranslate/UP26384/T1/P2/l/microsoft/computer_translation.html?wl_lp=ZH_CN-EN&wl_fl=0&wl_rurl=http%3A%2F%2F%s%2F&wl_url=%s&wlg_table=-3
Find acronym (commonly used abbreviation, e.g. RTFM):
acronymfinder|http://www.acronymfinder.com/af-query.asp?String=exact&Acronym=%1
Search word meaning on urbandictionary.com (slang dictionary, acronyms etc.):
urbandictionary|http://www.urbandictionary.com/define.php?term=%s

3) Dictionary reference

Rambler Dictionary - translation words.
To translate a word type "eng2rus word" or "rus2eng word".
eng2rus|http://www.rambler.ru/dict/scripts/relay-enru.cgi?query=%s&where=enru&start=0&count=15
 Для использования наберите в консоли "rus2eng слово"
rus2eng|http://www.rambler.ru/dict/scripts/relay-ruen.cgi?query=%s&where=ruen&start=0&count=15
Info-portal of Russ Portal Company.
To reference the encyclopaedia type "? word".
?|http://www.rubricon.com/qe.asp?qtype=1&id=0&srubr=0&fstring=%s
Lingvo translator at Yandex.ru.
Type "translate russian_word" to translate from Russian into English. Type "translate english_word" to translate from English into Russian.
translate|http://lingvo.yandex.ru/cgi-bin/lingvo.pl?text=%s

4) For web-designers.

W3 Schools contains detailed info on html tags.
To get info on a specific tag, just type "tag tag".
tag|http://www.w3schools.com/tags/tag_%s.asp?output=print
W3 Schools validator for check your site:
validator_html|http://validator.w3.org/check?uri=http%3A%2F%2F%s
validator_css|http://jigsaw.w3.org/css-validator/validator?uri=http%3A%2F%2F%s&usermedium=all

5) Web addresses without www. or http://

This alias saves you typing http:// and www.
Just use this pattern: "` amazon.com" and the browser window will become active after start.
`|C:\Program Files\Internet Explorer\IEXPLORE.EXE
The following alias functions in the same way, but the browser window opens on the background. It could be handy if you want to open a URL while browsing in another window.
``|C:\Program Files\Internet Explorer\IEXPLORE.EXE|||5
Certainly, you may substitute the browser path in these aliases for your favourite one.

5) Virtual folders

To open Windows virtual folders (e.g. My Documents or Control Panel) you should use their special identification numbers instead of names:
my_computer|explorer|::{20D04FE0-3AEA-1069-A2D8-08002B30309D}
my_network_places|explorer|::{208D2C60-3AEA-1069-A2D7-08002B30309D}
my_documents|explorer|::{450D8FBA-AD25-11D0-98A8-0800361B1103}
recycle_bin|explorer|::{645FF040-5081-101B-9F08-00AA002F954E}
printers_and_faxes|explorer|::{2227A280-3AEA-1069-A2DE-08002B30309D}
network_connections|explorer|::{7007ACC7-3202-11D1-AAD2-00805FC1270E}
