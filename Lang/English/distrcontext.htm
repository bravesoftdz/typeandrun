Distributed package

Contains the following:

TypeAndRun.exe - executed file
Lang\English.chm - english help file
Lang\Russian.chm - russian help file
Lang\English.lng - english language module
Lang\Russian.lng - russian language module

Files generated by the program:

Config.ini - alias configuration file
~Config.ini - removed dead alias configuration file
TypeAndRun.ini - program settings
History.ini - command history file
Plugins.ini - plugins configuration file

Plugins\ - default plugins directory
