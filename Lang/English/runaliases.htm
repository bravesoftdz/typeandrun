How to use the console

Push the hot key combination for the console and enter an aliases in one of the following acceptable forms:

<alias> [parameters] start an alias with parameters
<sysalias> [parameters] start an system alias with parameters
<plugin_alias> [parameters] start an plugin alias with parameters
<path-to-folder> open the folder with your File Manager
<path-to-file> open the document
www.* open the www.* address
http://* open the http://* address
*@*.* write an e-mail message to the *@*.* address
/about show the About dialog box
/checkconfig вернуть мёртвые алиасы в конфиг и запустить проверку конфига на мёртвые алиасы
/config start the Config Editor
/exit exit the TypeAndRun program
/help execute help file
/hide hide the console
/homedir open TypeAndRun directory
/inidir открыть директорию с используемым в данный момент конфигов алиасов
/inifiles shows *.ini files paths that program uses
/load [plugin_name] load plugin into memory
/reread reread all configuration files
/settings show the TypeAndRun settings
/show показать окно консоли (звучит бредово, но может использоваться из плагинов, например, из tar_hotkeys)
/unload [plugin_name] unload plugin from memory
<any other string> execute the command specified through the plugin or command shell
