Command line parameters

Usage:

TypeAndRun.exe [--settings=path] [--config=path] [--configold=path] [--history=path] [--plugins=path] [--add=path] [--exec=string]

Where:

--settings=path - specify file with settings. If no --settings parameter given than TypeAndRun uses TypeAndRun.ini from the TypeAndRun folder. Path - full path to the *.ini file. If only name given then using TypeAndRun folder as a path to the file. Parameter ignores if TypeAndRun already runs.

--config=path - specify file with config of aliases. If no --config parameter given then TypeAndRun uses Config.ini from the TypeAndRun folder. Path - full path to the *.ini file. If only name given then using TypeAndRun folder as a path to the file. Parameter ignores if TypeAndRun already runs.

--configold=path - specify file with config of dead aliases. If no --configold parameter given then TypeAndRun uses file from the TypeAndRun folder with the same name (with '~' prefix). Path - full path to the *.ini file. If only name given then using TypeAndRun folder as a path to the file. Parameter ignores if TypeAndRun already runs.

--history=path - specify history file. If no --history parameter given then TypeAndRun uses file History.ini from the TypeAndRun folder. Path - full path to the *.ini file. If only name given then using TypeAndRun folder as a path to the file. Parameter ignores if TypeAndRun already runs.

--plugins=path - specify file with plugins settings. If no --plugins parameter given then TypeAndRun uses file Plugins.ini from the TypeAndRun folder. Path - full path to the *.ini file. If only name given then using TypeAndRun folder as a path to the file. Parameter ignores if TypeAndRun already runs.

--add=path - adds alias to the config. If no --add parameter given then no aliases than aliases not adds. Path - full path to the file or folder. If no Path parameter given, than aliases not adds. Parameter works even TypeAndRun already runs.

--exec=string - run string "string" via console.

--multiusers=[1\0] - указывает программе, в каком режиме работать с конфигами. Если указать 0, то будут использоваться конфиги из директории TypeAndRun. Если 1, то используются конфиги из директории %APPDATA%\TypeAndRun. Если же параметр --multiusers не указан, то используются настройки TypeAndRun из реестра.

If path or string contains spaces than you should type braces around it.

You can specify parameters in any order.

Warning! Incorrect usage of parameters may cause errors in config or even losing configs! Save you *.ini files before experiments with command line switches.
